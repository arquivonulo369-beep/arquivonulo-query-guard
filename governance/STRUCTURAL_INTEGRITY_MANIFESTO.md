# STRUCTURAL INTEGRITY MANIFESTO
ArquivoNulo Global

QueryGuard is not a logging utility.
It is not a wrapper.
It is not an API extension.

It is a deterministic governance layer.

Principles:

1. Governance precedes execution.
2. Canonical constraints precede agent autonomy.
3. Cryptographic verifiability precedes trust.
4. Deterministic logs precede narrative claims.
5. Authority derives from repository root state.

Integrity Model:

- Merkle-based inclusion proofs (SHA3-384)
- Ed25519 signature validation
- Snapshot reproducibility
- Ledger immutability

No authority is declared outside cryptographic verifiability.

This repository constitutes the canonical integrity reference for QueryGuard v1.0.0 baseline.
